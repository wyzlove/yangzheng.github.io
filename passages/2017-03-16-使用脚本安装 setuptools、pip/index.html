使用脚本安装 setuptools、pip

创建 ez_setup.py、 get-pip.py

setuptools：https://bootstrap.pypa.io/ez_setup.py
pip：https://bootstrap.pypa.io/get-pip.py
以上，均手动复制粘贴并新建脚本文件为： ez_setup.py、 get-pip.py

安装 setuptools

打开cmd，cd到 ez_setup.py文件目录，输入 python ez_setup.py install，自动安装即可!
安装完成后，在python的Scripts文件夹下的目录如下：

为easy_install设置环境变量

至此，即可在cmd中直接使用easy_install。如安装django: easy_install django;安装numpy： easy_install numpy（需保持联网，可能因为网速原因，可能会失败，建议多试几次）

安装pip

和 setuptools同理， 打开cmd，cd到 get-pip.py 文件目录，输入 python get-pip.py，自动安装即可，安装完成后在python的 Scripts文件夹下的目录如下：（可在命令行中输入pip，如果出现pip的用法介绍，说明安装成功）

为pip设置环境变量

至此，即可在cmd中直接使用pip。如安装 symp : pip install symp ;安装matplotlib： pip install symp （需保持联网）
